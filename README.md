# ALFA_CLOWN_COMPRABUSINESS
alfa_clown_comprabusiness 
Prima Release

#### Come installare:
0. Scarica ed estrai
1. Ficcalo nella cartella delle resources
2. Ficca ensure alfa_clown_comprabusiness nel server.cfg

### Dipendenze:
* Gridsystem (obbligatorio)
* okokTextUI (facoltativo, se non lo hai cambia il codice)
* ESX (LEGACY)
